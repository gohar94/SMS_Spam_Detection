Potential large-scale uses

This app can help in detecting if an SMS/Whatsapp message is spam or not. It can be used to consume messages in smartphones before sending the user a notification so that spam messages don't caause disturbance and don't show up in the inbox.

Overview

A classifier has been trained using a dataset (available here) which predicts whether a given SMS message is spam or not based on what it has learnt from the annotated dataset.
The dataset has room for improvement and catering to many more cases; this has been left as future work.

Technologies Used

This classifier that powers this app has been built using the Microsoft Azure Machine Learning API.
